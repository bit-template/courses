## Course Types

| Type        | Description                                           |
| ----------- | ----------------------------------------------------- |
| Mandatory   | Must be completed before contributing to the project. |
| Recommended | Strongly encouraged but optional.                     |
| Optional    | Useful for advanced or specialized roles.             |

## Naming Convention

```
<Category>-<Sequence>

Examples:

GIT-001
PY-001
LNX-001
SEC-001
DOC-001
```

---

## Adding a New Course

1. Assign a unique Course ID.
2. Choose the appropriate category.
3. Fill in all required fields.
4. Commit the changes.
5. Update the learning path (if applicable).

---

## Removing a Course

1. Delete the corresponding row from the table.
2. Remove references from any learning paths.
3. Notify contributors if the course was previously mandatory.
